# Brand Soul — template

> **How to use this file.** This is a fill-in-the-blank template, not finished copy. Work top to bottom; each section has a prompt in _italics_ and a short example in `code` to calibrate tone. Delete the guidance once you've written the real thing. Aim for decisions, not adjectives — "we sound like a sharp friend who's done this before" beats "we're innovative and passionate." Revisit quarterly.

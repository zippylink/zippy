# Costs

**~$0/month at MVP scale.** Every paid integration is env-gated to a no-op — a fresh clone boots on an empty `.env.local`, and each feature turns on only when you add its key. The only day-one costs are **AI tokens** (no free tier — you pay from the first token) and, if you want it, **Mobbin's MCP** (~$10/mo, paywalled). Everything else has a free tier that comfortably covers an MVP.

## The two real day-one costs

- **AI (OpenAI / Anthropic): NO free tier.** Unlike everything else here, there's no "boots for free" path — the moment `@stack/ai` makes a call with your `AI_API_KEY`, you pay per token. Cost depends entirely on model + volume; pick a cheaper model id to cap it. Details + provider switching: [`ai.md`](./ai.md).
- **Mobbin MCP: paywalled.** The `mobbin` server in `agents/mcp.json` gives your agent 600k+ real app UI screens as design reference, but it requires a paid Mobbin plan (~$10–13/mo). Without it the server returns a 401 — drop it from `.mcp.json` if you don't want it; everything else works.

## Everything else is genuinely free at MVP scale

Postgres (Neon), frontend hosting (Cloudflare Workers), secrets (Infisical), email (Resend, within 100/day), analytics (PostHog + Clarity), and payments (Creem — you only pay when you make a sale) all sit inside free tiers or pay-per-use that an early-stage project rarely exceeds. And because every one is **env-gated**, you don't pay — or even sign up — until you actually want that feature on.

These free tiers aren't just MVP-training-wheels — most of them carry you **well into real scale**. 100k Cloudflare requests/day, Neon's compute allotment, Infisical for a small team, 1M PostHog events/month, Clarity uncapped forever: that's a live product with real users, not a toy. And when you do outgrow one, the next tier is a card, not a rebuild — Cloudflare Workers jumps to $5/mo, Neon and PostHog go usage-based, Resend Pro is $20. You add a line item as a specific ceiling actually bites, one at a time — you don't front-load a bill to launch.

## Compare: Creem vs Stripe (the MoR premium)

Creem is a **Merchant of Record** — it's the seller of record, so it handles global sales tax / VAT for you. That convenience costs more than a raw processor: **3.9% + $0.40/txn** vs **Stripe's 2.9% + $0.30**. That ~1% + $0.10 delta is the price of not doing tax compliance yourself. Optional Creem features (revenue splits, affiliates, cart recovery) add further percentage points — check current rates before enabling them. Payments route through the `@stack/payment` adapter, so swapping Creem for Stripe/Paddle/Lemon Squeezy is a one-file change.
